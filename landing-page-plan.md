## 1. User Journey (App Flow)

Tujuan utama landing page ini adalah mempermudah orang memesan via WhatsApp (karena di gambar tertera nomor kontak).

1. **Landing:** User datang dan disambut visual Bakpia yang menggugah selera.
2. **Explore:** User scroll ke bawah untuk melihat varian rasa dan harga.
3. **Trust Building:** User melihat sertifikasi Halal dan program sosial (donasi panti).
4. **Action:** User klik tombol "Pesan Sekarang" pada varian yang disukai.
5. **Conversion:** Sistem mengarahkan user ke WhatsApp dengan pesan otomatis: *"Halo Bakpia-Bumurto, saya ingin pesan varian [Nama Varian]..."*

## 3. Rekomendasi Visual & Grafis

Untuk membuat landing page ini terlihat profesional, kita perlu menjaga konsistensi visual:

* **Palet Warna:**
* **Primary:** *Forest Green* (diambil dari warna kemasan).
* **Secondary:** *Cream/Light Yellow* (untuk kesan hangat dan warna kulit bakpia).
* **Accent:** *Deep Purple* (untuk ubi ungu) dan *Chocolate Brown*.


* **Typography:** Gunakan font *Sans Serif* yang bersih untuk navigasi, dan font *Serif* atau *Script* yang elegan untuk headline agar terasa seperti makanan premium.
* **Efek Grafis:**
* **Parallax Scrolling:** Foto bakpia di section Hero sedikit bergerak saat di-scroll.
* **Hover Effect:** Saat kursor diarahkan ke foto menu (misal: Ubi Ungu), foto sedikit membesar (*zoom in*) dan muncul deskripsi singkat.
